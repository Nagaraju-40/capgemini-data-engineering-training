# 📊 SQL Data Cleaning & Transformation Project

---

## 📌 Problem Summary

The dataset used in this project contained several data quality issues:

* Missing (`NULL`) values affecting calculations
* Inconsistent text formats (uppercase, lowercase, extra spaces)
* Date columns requiring extraction and comparison
* Lack of categorization for better analysis

To address these issues, the task involved:

* Handling NULL values effectively
* Cleaning and standardizing text data
* Extracting insights from date columns
* Applying conditional logic for classification

---

## ⚠️ Key Challenges

* Managing NULL values without impacting calculations
* Selecting appropriate string functions for cleaning data
* Handling inconsistent text formats across records
* Understanding date functions and calculating differences correctly
* Applying accurate conditional logic for categorization

---

## 🛠️ Approach

* Identified columns with missing and inconsistent data
* Applied NULL-handling techniques to avoid errors in computations
* Cleaned and formatted text data for consistency
* Used date functions to extract components and calculate differences
* Implemented conditional logic to categorize and structure data

---

## 🔄 Key Transformations Used

### 🔸 NULL Functions

* Replaced NULL values with default values
* Ensured calculations do not return NULL
* Enabled safe comparisons

**Importance:**
Prevents incorrect results in aggregations and improves data reliability

---

### 🔸 String Functions

* Converted text to uppercase/lowercase
* Removed extra spaces and unwanted characters
* Extracted specific parts of strings
* Standardized text formatting

**Importance:**
Improves readability and consistency of data for reporting

---

### 🔸 Date Functions

* Extracted year, month, and other components
* Calculated differences between dates
* Used current date for comparisons
* Reformatted date values

**Importance:**
Supports time-based analysis and decision-making

---

## 📚 Learnings

* Importance of data cleaning before analysis
* Practical handling of NULL values in real datasets
* Effective use of string functions for data standardization
* Role of date functions in deriving insights
* Application of conditional logic for categorization

---

## 📊 Output

After applying transformations:

* ✅ Clean dataset with properly handled NULL values
* ✅ Consistent and standardized text data
* ✅ Accurate date-based calculations
* ✅ Categorized and structured data for analysis
